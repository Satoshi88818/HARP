This is **HARP v7 — H&L (Healthcare & Longevity)**. 

This version integrates the high-assurance foundation of v6 with a new **Mission Context** system. It allows the platform to pivot between "Rescue Mode" (high-power, triage-centric) and "Care Mode" (low-power, tactile-sensitive, consent-driven).

### Key New Improvements in v7:
*   **Imp-30 (Tactile Sensitivity):** `PressureMap` integration and `BruisePrevention` logic in the WBC.
*   **Imp-31 (Consent Gate):** Assist is blocked in Care Mode unless `consent_status` is `Explicit`.
*   **Imp-32 (Clinical Vitals):** `SubjectEvidence` now consumes heart rate, SpO2, and agitation metrics to modulate robot speed.
*   **Imp-33 (Social HRI):** Soft-light breathing patterns (SoftGreen, NightGuidance) for non-distressing interaction.
*   **Imp-34 (Privacy Scrubber):** Telemetry entries are "scrubbed" of high-resolution biometric identifiers before flash-journaling to meet HIPAA-like standards.

### 2. src/control/wbc.rs (Excerpts)
```rust
//! # HARP v7 — `control/wbc.rs`
//! 
//! Imp-30: Bruise Prevention & Micro-force compliance.

impl ImpedanceController {
    /// Extended for Healthcare: modulates torque to stay under kPa thresholds.
    pub fn apply_tactile_safety(
        &mut self,
        cmd: &mut [f32; DOF],
        pressure: &PressureMap,
        mode: MissionMode,
    ) {
        if mode == MissionMode::Healthcare {
            for &kpa in pressure.contact_points_kpa.iter() {
                if kpa > CARE_PRESSURE_LIMIT_KPA {
                    let excess = kpa - CARE_PRESSURE_LIMIT_KPA;
                    // Exponential back-off for micro-forces
                    for val in cmd.iter_mut() {
                        *val *= (1.0 - (excess / 10.0)).clamp(0.1, 1.0);
                    }
                    defmt::warn!("Tactile Safety: Pressure {=f32}kPa exceeds care limit", kpa);
                }
            }
        }
    }
}

// Kani Harness for Care Mode
#[cfg(kani)]
#[kani::proof]
fn verify_bruise_prevention_limit() {
    let mut cmd = [10.0f32; DOF];
    let mut pressure = PressureMap::default();
    pressure.contact_points_kpa[0] = 25.0; // High pressure
    
    let mut imp = ImpedanceController::new();
    imp.apply_tactile_safety(&mut cmd, &pressure, MissionMode::Healthcare);
    
    // Command must be significantly attenuated
    assert!(cmd[0] < 5.0);
}
```

```rust
impl EthicalSafetyEngine {
    pub fn update_safety_assessment(
        &mut self,
        evidence: &SubjectEvidence,
        eskf_uncertainty: f32,
    ) -> SafetyDecision {
        
        // ---- Gate A: ESKF/PUF (From v6) ----
        if eskf_uncertainty > 0.15 || !evidence.puf_audit_ready {
            return SafetyDecision::Standby;
        }

        // ---- Gate B: Consent (Imp-31) ----
        if evidence.mission_context == MissionMode::Healthcare {
            match evidence.consent {
                ConsentStatus::Withdrawn => return SafetyDecision::Standby,
                ConsentStatus::Unknown => {
                    defmt::info!("Care: Awaiting explicit consent");
                    return SafetyDecision::Standby;
                },
                _ => {} // Explicit or Implied
            }
        }

        // ---- Gate C: Vitals Slowdown (Imp-32) ----
        let agitation_slowdown = 1.0 - (evidence.vitals.agitation_index * 0.5);
        
        // Logic continues to Shadow Monitor and primary belief...
        // [Existing v6 logic]
        SafetyDecision::Assist
    }
}
```

### 6. src/main.rs (The Mission-Aware Loop)
```rust
#[task(binds = TIM1_UP, priority = 3, shared = [hal, safety, dreamer, system_mode, ...])]
fn reflex_loop(mut cx: reflex_loop::Context) {
    // 1. Determine Mission Context (External input or hard-coded per deployment)
    let mission = MissionMode::Healthcare; 

    // 2. Read Physical Constraints (Pressure + Torque)
    let (physics, pressure) = cx.shared.hal.lock(|hal| {
        (hal.read_physical_constraints(), hal.read_pressure_map())
    });

    // 3. Subject Evidence with Consent
    let evidence = get_evidence_with_vitals_and_consent();

    // 4. Safety Engine (now with Consent Gate)
    let decision = cx.shared.safety.lock(|s| {
        s.update_safety_assessment(&evidence, 0.01)
    });

    // 5. Motion Planning (Dreamer v7)
    let mut motion = cx.shared.dreamer.lock(|d| {
        if mission == MissionMode::Healthcare {
            d.plan_care_task(&evidence) 
        } else {
            d.plan_rescue_task(&evidence)
        }
    });

    // 6. WBC with Tactile Bruise Prevention (Imp-30)
    cx.shared.impedance.lock(|imp| {
        imp.apply_tactile_safety(&mut motion, &pressure, mission);
    });

    // 7. Write Commands
    cx.shared.hal.lock(|hal| hal.write_commands(motion));
}
```
